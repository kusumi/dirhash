NAME
       dirhash

SYNOPSIS
       dirhash [options]... [paths]...

DESCRIPTION
       Recursively walk directory trees and print message digest of regular
       files.

OPTIONS
       -abs
           Print file paths in absolute path.

       -debug
           Enable debug print.

       -h
           Print usage and exit.

       -hash_algo <string>
           Hash algorithm to use.  Defaults to "sha256".

       -hash_only
           Do not print file paths.

       -hash_verify <string>
           Message digest to verify in hex string.

       -ignore_dot
           Ignore entries start with '.'.

       -ignore_dot_dir
           Ignore directories start with '.'.

       -ignore_dot_file
           Ignore files start with '.'.

       -ignore_symlink
           Ignore symbolic links.

       -lstat
           Do not resolve symbolic links.

       -squash
           Print squashed message digest instead of per file.

       -v
           Print version and exit.

       -verbose
           Enable verbose print.
